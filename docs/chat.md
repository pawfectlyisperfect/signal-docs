# Chat

Signal has two chat channels: group chat and squad chat. Both show up through the same Chat tab UI, with squad messages tagged and colored so you can tell them apart from group-wide messages at a glance.

## Group chat vs squad chat

Group chat goes to everyone in the group. Squad chat only goes to your squad and any linked squads. Which one you're posting to depends on which tab you're in (Group Chat vs Squad Chat), they're not merged into one box you have to filter.

## Muting

Anyone with the Member Mute permission (Officer and above) can mute a player. A muted player's messages get silently dropped server-side, not just hidden client-side, so there's no way around it by reconnecting or anything like that. Muting is indefinite until someone with permission unmutes, there's no auto-expiry.

Muted players also can't use the comm wheel. Same enforcement, same reasoning: if you can't talk, you can't ping either.

Mutes show up in the Player Manager tab alongside bans, with unmute/unban actions available to whoever has permission.

## Chat oversight

Someone with full member management permissions can toggle chat oversight for themselves, which lets them see squad chat from squads they're not actually in. This is meant to be transparent, not sneaky, everyone in the group can see who currently has oversight turned on, it's not a silent admin power.

## Notifications for chat

Standout roles (Officer and above) pinging in chat can trigger a notification if you've got chat notifications turned on in Settings, using the same notification system as everything else in the mod rather than a separate toast.
